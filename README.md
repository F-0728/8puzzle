# 8 puzzle on Web

## Run the dev server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## GitHub Pages

Check [https://f-0728.github.io/8puzzle/](https://f-0728.github.io/8puzzle/).
(Mobile displaying is not supported yet)

## Features

- Auto shuffling
- Auto counting min steps (using A* search algorithm)
- Step counter

## Tech Stack

- Next.js
- React
- Chakra UI
